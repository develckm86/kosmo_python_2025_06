# 파이썬의 기본형 관점
자바처럼 `primitive` 키워드는 없지만 숫자·불리언·문자열·None·바이너리가 가장 기초적인 빌트인 타입입니다. 모두 객체라서 메서드·연산을 바로 쓸 수 있습니다.

## 1. 숫자형: `int`, `float`, `complex`
- `int`: 크기 제한 없는 정수, 밑줄로 가독성 향상 `1_000_000`.
- `float`: IEEE 754 배정밀도(근사). 오차 보정은 `decimal` 고려.
- `complex`: 복소수 `a + bj` 형식. `j`는 허수 단위.
- 진법 리터럴: `0b1010`, `0o17`, `0xFF`.
- 주요 연산
  ```python
  a, b = 10, 3
  a + b, a - b, a * b
  a / b, a // b, a % b, a ** b
  abs(-5), round(3.1415, 2), divmod(10, 3)
  ```
- 부동소수 오차 예시
  ```python
  0.1 + 0.2          # 0.30000000000000004
  0.1 + 0.2 == 0.3   # False
  ```

## 2. 불리언: `bool`
- 값은 `True`, `False`. 비교/논리 연산 결과로 생성.
- `bool`은 `int`의 서브클래스(`True == 1`), 의미상 구분해서 사용.
- truthy/falsey: `0`, 빈 시퀀스(`[]`, `""`, `{}`), `None` 등은 거짓으로 평가.

## 3. 문자열: `str`
- 불변(immutable) 유니코드 시퀀스. `'`, `"`, `"""` 모두 사용.
- 기본 연산: 덧셈(연결), 곱셈(반복), 슬라이싱 `s[0:3]`, 길이 `len(s)`.
- 자주 쓰는 메서드: `upper/lower`, `strip`, `replace`, `split`, `" ".join(list)`, `find`, `startswith/endswith`, `count`.
- 포매팅: `f"안녕 {name}"`(권장), `"{} {}".format(a, b)`, `%` 방식(구식).
- 멤버십: `"py" in "python"` → `True`.

## 4. None
- 값이 없음을 나타내는 단일 객체.
- 비교는 `is` 사용: `if value is None:`.
- 함수가 반환을 명시하지 않으면 기본 `None`.

## 5. 바이너리: `bytes`, `bytearray`
- `bytes`: 불변, `b"hi"`나 `bytes([104, 105])`.
- `bytearray`: 가변.
- 인코딩/디코딩
  ```python
  b = "한글".encode("utf-8")
  s = b.decode("utf-8")
  ```

## 6. 연산자 요약
- 산술: `+ - * / // % **`
- 비교: `== != > >= < <=`
- 논리: `and`, `or`, `not` (단락 평가)
- 멤버십: `in`, `not in`
- 동일성: `is`, `is not` (`None` 체크에 사용)
- 비트: `& | ^ ~ << >>`
- 할당/복합: `=`, `+=`, `-=`, `*=`, `/=`, `//=`, `%=`, `**=`

## 7. 자바와 비교
- primitive/Wrapper 구분 없이 **모두 객체**. 박싱/언박싱 걱정 없음.
- 정수 오버플로 없음. `/`는 항상 실수, 정수 나눗셈은 `//`.
- `char` 없음. 문자열은 모두 유니코드, 한 글자도 길이 1의 문자열.
- 문자열 값 비교에 `==` 사용 가능. 동일성 비교는 `is`.
- `null` 대신 `None`, 비교는 `is None`.

## 8. 연습 아이디어
- 각 타입으로 값 하나씩 만들고 `type()` 확인.
- `/`, `//`, `%`, `divmod` 차이 실습.
- `and`/`or` 단락 평가로 기본값 선택: `value = user_input or "default"`.
- 포매팅으로 소개 문구 만들기, 같은 내용을 자바 스타일로 비교해보기.

## 9. 참고: 허수 직관
- `i`는 `i² = -1`이 되도록 정의한 허수 단위, 파이썬에서는 `j` 사용.
- 파동·회전·신호 처리에서 크기와 위상을 함께 표현할 때 유용하다.
