html을 꾸며주는 양식 또는 문서

CSS Selector(선택자) 요소간의 관계를 표현하는 것

  • div = div 요소 선택
  • div.red = class가 red인 div 요소 선택(class는 여려개 존재)
  • #query = id가 query인 요소 선택(id는 보통 html 문서에 1개만 존재)
  • form input = form 요소 안에 있는 모든 자손(자식의 자식의..)인 input
  • form>input = form 요소의 자식 요소인 input
  • input[type="text"] = input 요소인데 속성인 type이 text인것
  • :nth-child(n) : 몇번째 요소
  • div,li or .red,.blue,.color = 여러요서를 동시에 선택
  • div+p = div 요소에 인접한 형제 p 요소
  • div~p = div 요소의 형제들 중에 p 요소

URL uniform resource locator 리소스(찾을 수 있는 자원)의 주소

https://comic.naver.com/webtoon/detail?titleId=827319&no=1&week=mon

  • https : http 통신으로 리소스를 요청하겠다.
  • comic.naver.com : (도메인) 서버의 주소 IP
  • webtoon/detail : 패스, 리소스의 위치
  • ?titleId=827319&no=1&week=mon : 파라미터 페이지를 동적으로 만들어 준다.
  • ? : 파라미터 시작, & : 파라미터 구분자, KEY=VALUE
